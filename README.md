# 🏦 Bank Management System with User Permissions (C++)

A console-based **Bank Management System** developed using **C++** that simulates a real banking environment.

The system allows administrators to manage bank clients, perform financial transactions, manage system users with different permission levels, and keep transaction logs using file handling.

---

## 📌 Features

### 👥 Client Management
- ➕ Add New Client
- 📋 View Clients List
- 🔍 Find Client
- ✏️ Update Client Information
- ❌ Delete Client

### 💳 Transactions
- 💰 Deposit
- 💸 Withdraw
- 🔄 Transfer Between Clients
- 📜 Transfer Log
- 💵 View Total Balances

### 👤 User Management
- 🔐 Secure Login System
- ➕ Add New User
- 👥 View Users List
- 🔍 Find User
- ✏️ Update User
- ❌ Delete User
- 🛡️ User Permissions System

### 💾 Data Management
- File Handling
- Persistent Data Storage
- Transaction Logging

## 🔐 User Permissions

The system implements a permission-based authentication system.

Permissions can be assigned to users to control access to:

- Show Clients
- Add Client
- Delete Client
- Update Client
- Find Client
- Transactions
- Manage Users
- Full Access
